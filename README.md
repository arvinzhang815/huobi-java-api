# huobi-java-api

### How configure unit test

1. make dir  `src/test/resources/`, make dir as Test Resource Root
2. create file `config.properties`.
    
    ``` properties
    #sample
    #apiKey=e8d23f3b-92721ba8-7b804c0f-xxxx
    #apiSecret=bd14ad47-fe8f8ed9-bf3e32df-xxxx
    ```
